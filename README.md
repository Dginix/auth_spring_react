# auth_spring_react

Authentication sample app with [Spring Boot](http://projects.spring.io/spring-boot/) on server side and [React](https://reactjs.org/) on client side.

## Requirements

Back-end:
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- PostgreSQL

Front-end:
- React
- react-router-dom
- axios
- MUI
- react-hook-form
- yup
